## Reporting a Vulnerability

We take security vulnerabilities seriously and appreciate your help in keeping our users safe. If you discover a security vulnerability in Grok AI Integration, please report it responsibly.

### How to Report

**Please DO NOT report security vulnerabilities through public GitHub issues.**

Instead, please report security vulnerabilities by emailing:
- **security@ndestates.com**

When reporting, please include:
- A clear description of the vulnerability
- Steps to reproduce the issue
- Potential impact and severity assessment
- Any suggested fixes or mitigations (optional)
- Your contact information for follow-up

### Response Timeline

- **Initial Response**: We will acknowledge receipt of your report within 48 hours
- **Investigation Update**: We'll provide a more detailed response within 7 business days indicating our next steps
- **Resolution Update**: We'll keep you informed of our progress every 7-14 days depending on complexity
- **Final Resolution**: We'll notify you when the issue is resolved or if we've decided not to proceed

### What to Expect

#### If Your Report is Accepted:
- We'll work with you to understand and validate the vulnerability
- We'll develop and test a fix
- We'll coordinate a public disclosure date with you
- You'll receive credit in our security advisory (unless you prefer anonymity)
- We'll release a security update with the fix

#### If Your Report is Declined:
- We'll provide a clear explanation of why we declined the report
- This might be because:
  - The issue doesn't meet our definition of a security vulnerability
  - The issue is already known and addressed
  - The issue affects unsupported versions only
  - The issue requires user interaction or has minimal impact
- We'll still appreciate your effort in helping improve our security

### Responsible Disclosure

We kindly ask that you:
- Give us reasonable time to fix the issue before public disclosure
- Avoid accessing or modifying user data
- Don't perform DoS attacks or degrade service performance
- Don't spam our systems with automated vulnerability scanners

### Security Updates

Security updates will be released as patch versions with high priority. All supported versions listed above will receive security updates.
